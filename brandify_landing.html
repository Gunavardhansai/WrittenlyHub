# Brandify Studio Landing Page ## Overview This repository contains the HTML, CSS, and JavaScript code for the **Brandify Studio** landing page. It is a modern, clean, and visually appealing single-page website designed for a creative agency specializing in branding, digital marketing, and web/app development. ## Features - **Responsive Design**: Optimized for desktops, tablets, and mobile devices. - **Sticky Navigation**: Includes smooth scroll behavior and a shadow effect on scroll. - **Hero Section**: Prominent headline, introductory text, and quick quote form. - **About Section**: Company background and team image. - **Services Section**: Grid-based cards detailing services. - **Testimonials**: Client feedback with styling for emphasis. - **Contact Section**: Company contact details and inquiry form. - **Footer**: Quick navigation links and copyright notice. ## Technologies Used - **HTML5** for structure - **CSS3** with custom properties for styling and theming - **JavaScript (Vanilla)** for interactivity - **Google Fonts** (Inter, Playfair Display) - **Font Awesome** for icons ## Getting Started 1. Clone or download the repository. 2. Open `index.html` in any modern web browser. ``` git clone cd ``` 3. Double-click `index.html` or use a local server for preview. ## Customization - Update the `:root` CSS variables to change theme colors, font families, and spacing. - Replace images with your own by updating `img` tags. - Modify the `services` and `testimonials` sections to reflect your agency's offerings. ## Contact For any questions or contributions, reach out at **hello@brandifystudio.com**.
